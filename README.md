## 프로젝트 소개

각 플랫폼에 흩어져있는 공연(콘서트 및 페스티벌) 정보를 한눈에 볼 수 있는 서비스입니다. 공연 및 아티스트 알림 제공을 통해 공연을 놓치는 불편함을 해소하고자 합니다.


![프로젝트 소개](https://github.com/user-attachments/assets/0f0f13be-350b-431b-bf8a-bbbb08245141)



## 기술 스택

- Java 21
- Gradle
- Spring Boot 3.3.4
- Spring Data JPA
- Spring Security
- QueryDSL
- PostgreSQL
- AWS(ECS, RDS, S3)
